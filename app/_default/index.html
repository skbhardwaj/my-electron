INFO: This application is developed using Electron.
Electron (formerly known as Atom Shell) is an open-source framework developed and maintained by GitHub. Electron allows for the development of desktop GUI applications using front and back end components originally developed for web applications: Node.js runtime for the backend and Chromium for the frontend. Electron is the main GUI framework behind several notable open-source projects including Atom, Visual Studio Code, the Tidal client app, Light Table and Discord. [https://en.wikipedia.org/wiki/Electron_(software_framework)]
